# Ultra Car

Site do aplicativo Ultra Car


## Instalação


clone o repositorio
```bash
  git clone https://github.com/Lucas-Jav/UltraCar.git
```

entre na pasta raiz
```node
  cd UltraCar
```

instale as dependencias
```node
  npm install
```

instale o bun
```node
  npm install -g bun
```


## Rodando localmente

Inicie o servidor

```bash
  npm run dev
```

ou

```bash
  yarn dev
```

ou

```bash
  bun run dev
```


## Rodando com Docker

na pasta do projeto, rode este comando:

```docker
  docker-compose up --build --no-recreate -d
```
